# Echo App Center Connection Settings

Echo App Center connects to Echo App Server by using an IP/hostname and port.

Open:

```text
Settings > Server Connection
```

Set:

```text
Protocol: http
Server IP / Hostname: 192.168.0.50
Server Port: 8080
```

This builds the full URL:

```text
http://192.168.0.50:8080
```

Use the server computer's LAN IP, not `localhost`, when connecting from another computer.
