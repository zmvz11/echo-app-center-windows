# Final Build Checklist

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run final-check`
- Start Echo App Server
- Start Echo App Center local agent
- Open App Center UI
- Create first owner when needed
- Set server IP/hostname and port
- Test server and local agent connection
- Create an app in Admin Portal
- Upload media
- Upload and publish a release package
- Install the app from Store or Library
- Repair, update, launch, and remove the app from Library
- Confirm client check-in appears in Admin Portal
