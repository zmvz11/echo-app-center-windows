# Echo App Center — Windows

Clean GitHub-ready package for Echo App Center on Windows.

## Install

Run one installer from the repo root:

```text
INSTALL.bat
```

Do not run old helper scripts. This package intentionally has one obvious install entry point.

## After install

Windows installs Echo App Center as a normal app with Start Menu/Desktop shortcuts. Linux installs an AppImage and desktop launcher.

On first launch, enter the Echo App Server IP/hostname and port. If the server has no Owner account, App Center will show the first-admin setup page.
